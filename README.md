# Timesheet — Учёт рабочего времени

Проект для учёта отработанных часов по проектам и задачам с формированием отчётов для расчёта зарплаты.

## 📋 Описание

Timesheet — это система учёта рабочего времени, которая позволяет:
- Отслеживать время, затраченное на проекты и задачи
- Формировать отчёты за любые периоды для расчёта зарплаты
- Учёт времени в реальном времени через встроенный таймер
- Утверждать отчёты руководителями

## 🎯 Основные возможности

| Функция | Описание |
|---------|----------|
| **Таймер** | Учёт времени в реальном времени с возможностью паузы/стопа |
| **Отчёты** | Генерация отчётов за выбранный период по сотрудникам/проектам |
| **Проекты** | Управление проектами и задачами |
| **Утверждение** | Workflow утверждения отчётов руководителями |

## 🏗️ Архитектура данных

### Основные сущности

- **Сотрудник** — пользователь системы с ролью (Сотрудник/Руководитель)
- **Проект** — проект или задача, на которую ведётся учёт времени
- **Запись времени** — запись с датой, количеством часов и описанием выполненной работы

### Роли пользователей

| Роль | Возможности |
|------|-------------|
| **Сотрудник** | Замена времени, создание записей, просмотр своих отчётов |
| **Руководитель** | Все возможности сотрудника + утверждение отчётов подчинённых |

## 🚀 Установка

```bash
# Клонировать репозиторий
git clone https://github.com/ваш-юзернейм/timesheet.git

# Перейти в директорию проекта
cd timesheet

# Установить зависимости
pip install -r requirements.txt

# Настроить переменные окружения
cp .env.example .env
# Отредактируйте .env с вашими настройками

# Запустить приложение
python main.py
```

## 📖 Использование

1. **Регистрация** — создайте учётную запись как сотрудник или руководитель
2. **Создание проекта** — добавьте проекты/задачи для учёта времени
3. **Запуск таймера** — начните учёт времени в реальном времени
4. **Создание записей** — добавьте вручную отработанные часы
5. **Генерация отчёта** — выберите период и получите отчёт для зарплаты
6. **Утверждение** (для руководителей) — утвердите отчёты подчинённых

## 🛠️ Технологии

- Backend: Python, Flask/Django
- Frontend: React/Vue.js
- База данных: PostgreSQL/SQLite
- Таймер: WebSocket для реальновременного обновления
